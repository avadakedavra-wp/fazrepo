### Installation Instructions

#### After downloading a binary:
1. **Linux/macOS**: Make it executable with `chmod +x fazrepo`
2. **All platforms**: Move to a directory in your PATH (e.g., `/usr/local/bin` on Linux/macOS, or `%USERPROFILE%\.local\bin` on Windows)

#### After downloading a zip:
1. Extract the archive
2. Follow the README.txt inside for platform-specific instructions

## Build from Source

If you have Rust installed:

```bash
git clone https://github.com/avadakedavra-wp/fazrepo.git
cd fazrepo/apps/cli
cargo build --release
```

The binary will be at `target/release/fazrepo` (or `fazrepo.exe` on Windows).

## Verify Installation

After installation:

```bash
fazrepo --version
fazrepo check
```

## Troubleshooting

### Command not found?

1. **Restart your terminal** - PATH changes require a new session
2. **Check PATH**: Ensure the installation directory is in your PATH
3. **Manual PATH setup**:
   - **Linux/macOS**: Add `export PATH="$PATH:$HOME/.local/bin"` to `~/.bashrc` or `~/.zshrc`
   - **Windows**: Add `%USERPROFILE%\.local\bin` to your user PATH environment variable
